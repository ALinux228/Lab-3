# Задание 3
## Задача
### Решить с использованием последовательности задачу на обработку файлов/каталогов
Вывести пути ко всем файлам, не являющимся текстовыми (имеют расширение, отличное от txt), в указанном каталоге и его подкаталогах.
### Алгоритм решения
1. Написать рекурсивную функцию, обходящую все подкаталоги с помощью Directory.GetFiles и Directory.GetDirectories.
2. Проверять расширение каждого файла: если оно не .txt (без учета регистра), добавлять путь к результату.
3. Обрабатывать исключения (отсутствие доступа, ошибки ввода) с помощью блоков try with.
